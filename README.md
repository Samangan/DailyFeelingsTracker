Daily Feelings Tracker
----------------------

Buidling
========
`$ git clone https://github.com/Samangan/DailyFeelingsTracker.git`

`$ cd DailyFeelingsTracker`

`$ npm install`

`$ bower install`


Deploying Locally
=================

`$ grunt serve`


Testing
=======

`$ grunt`
